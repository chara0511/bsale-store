# Descripción General

BSale store es un ejercicio de postulación propuesto por la empresa [BSale](https://www.bsale.com.pe/) que consiste en crear una aplicación web, implementando cliente y servidor.
La aplicación es una tienda online que despliega productos; desde una base datos, agrupados por la categoría a la que pertenecen.
