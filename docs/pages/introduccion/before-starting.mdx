# Antes de empezar

## Diseño

**El diseño del producto es más importante de lo que se cree.** Apuntando a la simplicidad.

Este es el proceso de diseño que realicé para BSale store. Primero, empiezo con [Excalidraw](https://excalidraw.com/) para generar rápidamente maquetas de baja fidelidad (Wireframes).

![Excalidraw](/excalidraw.png)
